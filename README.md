# RestDelBot 5.0

AI-Powered Telegram bot for Rest Delivery estimate generation.

## Quick Start

1. Create virtual environment:
```bash
python -m venv venv
source venv/Scripts/activate  # Windows Git Bash
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Setup environment:
```bash
cp .env.example .env
# Edit .env with your credentials
```

4. Run:
```bash
python -m src.api.main
```

## Project Structure

- `src/` - Source code
- `config/` - Configuration
- `data/` - Data files
- `tests/` - Tests
- `logs/` - Log files
